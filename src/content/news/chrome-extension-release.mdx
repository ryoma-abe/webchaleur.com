## Chrome拡張機能「Site Launcher」をリリースしました 🚀

初めて開発したChrome拡張機能「Site Launcher」が、Google Chrome ウェブストアの審査を通過し、正式にリリースされました！

初めての拡張機能開発、そして審査プロセスということもあり、承認メールを受け取った瞬間は本当に嬉しかったです。

## Site Launcherとは？

**よく使うサイトに一瞬でアクセスできる、キーボード完結型のサイトランチャーです。**

### 主な特徴

- **高速アクセス**: `Cmd+M`（Windows: `Ctrl+M`）を押してから、登録したキーを押すだけ
- **カスタマイズ自由**: 好きなサイトに好きなキーを割り当て可能
- **開発者フレンドリー**: GitHub、Figma、Notionなど、よく使うツールを瞬時に開く

### こんな方におすすめ

- マウス操作を減らして作業効率を上げたい開発者
- 複数のWebツールを頻繁に切り替える方
- キーボードショートカットが好きな方

## 技術スタック

- React + TypeScript
- Vite（ビルドツール）
- Chrome Extension Manifest V3
- Tailwind CSS

## インストール方法

Chrome ウェブストアから無料でインストールできます：

[Site Launcher - Chrome ウェブストア](https://chromewebstore.google.com/detail/site-launcher/jahndejpknmaippmlngfodgkkmiodfai)
