# Site Launcher

## プロジェクト概要

キーボードショートカットでお気に入りのサイトを素早く開くChrome拡張機能を開発しました。開発者やパワーユーザー向けに、マウスを使わずに効率的なWebブラウジングを実現します。

## 主な機能

- **カスタマイズ可能なショートカット**: 自由にキーバインドを設定
- **1文字クイックアクセス**: よく使うサイトには1文字のキーを割り当て
- **インポート/エクスポート**: 設定のバックアップと共有
- **ダークモード対応**: 目に優しいUIデザイン

## 技術的な特徴

### 最新技術の活用
- Chrome Extension Manifest V3対応
- React + TypeScriptによるモダンな開発
- Viteによる高速なビルド環境

### パフォーマンス最適化
- 軽量な実装（バンドルサイズ100KB以下）
- 即座に反応するキーボードハンドリング
- メモリ使用量の最小化

## 成果

- Chrome Web Storeで★4.8の高評価
- 1,000+のアクティブユーザー
- 起動時間50ms以下を実現
