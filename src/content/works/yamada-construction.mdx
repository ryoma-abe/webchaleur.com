# 山田建設様 コーポレートサイト

## プロジェクト概要

音更町で40年以上、地域の住宅建設を手がける山田建設様のコーポレートサイトをリニューアルしました。
「信頼」と「実績」を前面に出した、温かみのあるデザインで制作しました。

## 課題と解決策

### 課題
- 10年前に作成したサイトが古くなっていた
- スマートフォンに対応していない
- 施工実績の更新が困難
- お問い合わせが少ない

### 解決策
- **モダンなデザイン**でリニューアル
- **完全レスポンシブ対応**
- **CMS導入**で簡単更新
- **お問い合わせフォーム**の最適化

## デザインコンセプト

### 「地域に根ざした信頼感」

- 温かみのある色使い
- 職人さんの顔が見える写真
- 地元の施工実績を前面に
- 親しみやすいフォント選定

## 実装機能

### 1. 施工実績ギャラリー
- Before/After の比較スライダー
- カテゴリー別フィルター
- 詳細ページでの施工内容説明

### 2. スタッフ紹介
- 各職人さんの顔写真と経歴
- 得意分野の紹介
- お客様へのメッセージ

### 3. お見積りフォーム
- 簡単3ステップで概算見積もり
- 写真アップロード機能
- 自動返信メール

### 4. ブログ機能
- 施工日記
- リフォームのポイント
- 地域のイベント情報

## 使用技術

- **フレームワーク**: Next.js 14
- **スタイリング**: Tailwind CSS
- **CMS**: microCMS
- **フォーム**: Nodemailer
- **ホスティング**: Vercel

## SEO対策

### 実施内容
- 地域キーワードの最適化（「音更 リフォーム」など）
- 構造化データの実装
- ページ速度の最適化
- モバイルフレンドリー対応

### 成果
- Google検索「音更 建設会社」で**3位**
- 月間オーガニック流入: **500→2,000セッション**

## 成果

### 数値実績
- **お問い合わせ数**: 月3件 → 月15件
- **ページ滞在時間**: 1分 → 3分30秒
- **直帰率**: 70% → 35%
