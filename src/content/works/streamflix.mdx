# StreamFlix

## プロジェクト概要

映画やドラマのストリーミング配信サービスを開発しました。ユーザーエクスペリエンスを重視し、高品質な動画配信と直感的なインターフェースを実現しています。

## 主な機能

- **高品質ストリーミング**: アダプティブビットレート配信
- **レコメンデーション**: 視聴履歴に基づく作品提案
- **マルチデバイス対応**: スマホ、タブレット、TV で視聴可能
- **オフライン視聴**: ダウンロード機能搭載

## 技術的な特徴

### フロントエンド

- React による高速な SPA 実装
- Progressive Web App 対応
- レスポンシブデザイン

### バックエンド・インフラ

- Node.js による API サーバー
- AWS CloudFront での CDN 配信
- MongoDB でのユーザーデータ管理

### 動画配信技術

- HLS/DASH 対応の動画配信
- 帯域幅に応じた自動画質調整
- DRM 保護によるコンテンツ保護

## 成果

- 同時接続数 10,000 人を実現
- 平均視聴時間が 2 時間/日を達成
- バッファリング率を 0.5%以下に削減
