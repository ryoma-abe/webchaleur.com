# おとふけカフェ様 ブランドサイト

## プロジェクト概要

音更町で人気のカフェ「おとふけカフェ」様のブランドサイトを制作しました。
お店の温かい雰囲気をWebでも表現し、来店前からワクワクする体験を提供します。

## コンセプト

### 「日常に、ちょっとした幸せを」

カフェの持つ温かみと、十勝の自然の恵みを感じられるデザインに仕上げました。

## 課題と解決策

### 課題
- SNSだけでは伝えきれない魅力
- 電話予約の対応が大変
- メニューの更新が頻繁
- イベント情報の告知方法

### 解決策
- **ブランドストーリー**の充実
- **オンライン予約システム**の導入
- **簡単メニュー更新**機能
- **イベントカレンダー**の実装

## デザインの特徴

### ビジュアル重視
- 料理の美味しさが伝わる大きな写真
- 店内の雰囲気が分かる360°ビュー
- スタッフの笑顔がいっぱい

### 色使い
- ナチュラルブラウンを基調
- アクセントに優しいグリーン
- 白を多用した清潔感

## 実装機能

### 1. オンライン予約
- カレンダーから簡単予約
- 席の指定も可能
- 予約確認メール自動送信

### 2. デジタルメニュー
- 季節限定メニューの表示
- アレルギー情報の記載
- カロリー表示

### 3. Instagram連携
- 最新の投稿を自動表示
- ハッシュタグでの投稿収集
- フォトギャラリー

### 4. ブログ機能
- 日替わりランチの紹介
- レシピ公開
- イベントレポート

## 技術仕様

- **CMS**: WordPress
- **予約システム**: カスタム開発
- **決済**: Square連携
- **SNS連携**: Instagram API
- **ホスティング**: エックスサーバー

## 成果

### 予約数の増加
- **導入前**: 1日平均5件（電話）
- **導入後**: 1日平均20件（オンライン15件、電話5件）

### SNS効果
- Instagramフォロワー: **500人→2,000人**
- 月間サイト訪問者: **1,000人→5,000人**

### 売上への影響
- 客単価: **10%アップ**
- リピート率: **60%→80%**

### プロの写真撮影
- 料理写真はプロカメラマンが撮影
- 店内の雰囲気を最大限に表現
- スタッフの自然な笑顔をキャプチャー

### モバイルファースト
