# Next.js 14 ISR Guide

ISR combines the best of static generation and server-side rendering.

## Key Features

- Static generation at build time
- Background regeneration at intervals
- Cache utilization during regeneration

## Implementation

Use the revalidate option to enable ISR:

```tsx
export const revalidate = 60; // Regenerate every 60 seconds
```

## Performance Comparison

- SSG: Fastest TTFB, minimal server load
- ISR: Fast TTFB, configurable updates
- SSR: Slower TTFB, real-time updates

## Summary

ISR is perfect for content that updates moderately often, providing both speed and freshness.
